# 🎲 Dice Simulator

A highly visual, probability physics simulation terminal script that renders multi-dice roll operations simultaneously using raw 2D matrix text graphics arrays.

---

## ✨ Features

* 📦 **Side-by-Side Graphic Layout Rendering:** Renders 2D matrix blocks horizontally using a row-by-row stitching index instead of vertical dumps.
* 📈 **Dynamic Parameter Sizing:** Supports scaling from 1 up to 6 concurrent dice drops using integrated list comprehensions.
* 🕒 **Immersive Shaking Animation:** Leverages timed stream write intervals to emulate real physical data collection states.
* 📊 **Roll Log Session Auditing:** Records complete batch arrays to evaluate frequency totals at run time.

---

## 🛠️ Tech Stack & Architecture

* **Language:** Python 3.x
* **Core Libraries utilized:** `random`, `sys`, `time`
* **Layout Design:** Custom structural mapping utilizing 2D array coordinates for ascii generation.
