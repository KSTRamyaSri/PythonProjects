# ✂️ Rock Paper Scissors Engine

A high-performance tournament console simulator utilizing dictionary key-value pair rule matrices to process runtime game decisions instead of nested conditional forks.

---

## ✨ Features

* 🧠 **Hash Map Rules Lookup Matrix:** Evaluates game outcomes in $O(1)$ constant time complexity using an isolated dictionary truth definition table.
* 📦 **Stitched 2D Horizontal Assets:** Concatenates separate vertical string arrays side by side inside the console layout buffer stream dynamically.
* 🕒 **Synchronized Interval Delay:** Implements countdown timers using iterative sleep flushes to mimic physical arcade game tension.

---

## 🛠️ Tech Stack & Architecture

* **Language:** Python 3.x
* **Core Libraries utilized:** `random`, `sys`, `time`
* **Design Pattern:** Data-driven programming (separating program game logic rules from the code execution loops).
